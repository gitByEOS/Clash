# ChangeLog

## v0.1.5

- Rust 版新增 `clash prompt` 捕获 Claude Code 实际请求提示词
- `clash prompt` 报告补齐本地规则、Skills 和延迟工具信息
- 移除 `clash prompt --text` / `--system`，保留 HTML 报告和 `--json`
- Windows 改为发布 Rust `clash.exe`，废弃 PowerShell 业务实现

## v0.1.4

- 支持追加系统提示词
- 增加通过`npm`安装以及管理`claude`版本

## v0.1.3

- 支持别名: `clash rename` 
- 支持参数穿透
- 自带statusline

## v0.1.2

- 多渠道支持

## v0.1.1

- TUI 底部截断修复
- 引导提示优化（DeepSeek 示例）

## v0.1.0

- 初始可安装版本
